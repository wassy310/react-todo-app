## Getting Started
- `git clone https://github.com/wassy310/react-todo-app`
- `cd todo-app`
- `npm install`
- `npm run dev`  
ターミナルに出力された[http://localhost:5173](http://localhost:5173/)をクリックするか`o`と入力すると、Reactアプリが立ち上がります
